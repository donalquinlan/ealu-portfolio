# ealu-portfolio

Single-page static portfolio for a Senior Technical Program Manager.

## Local preview

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Before you publish

1. Replace placeholder text in `index.html` (name, email, LinkedIn, location, experience).
2. Resume PDF is included as `resume.pdf` (sourced from your iCloud resume).
3. Push the repo to GitHub.

## Hosting on Spaceship

See [HOSTING.md](./HOSTING.md) for step-by-step instructions to publish at **portfolio.ealu.ai**.
